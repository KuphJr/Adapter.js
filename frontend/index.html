A Chainlink external adapter for fetching and processing text data



Fetch text from websites & APIs

Fetch of text directly from websites and APIs via HTTP "get" and "post" methods. Apply custom headers to access APIs which require authorization.

Apply JavaScript string methods

Fetched text can be processed with any combination of JavaScript string methods and regular expressions before being returned to a smart contract.


Available on Ethereum and Polygon

Hosted on a independently operated Chainlink nodes for the Ethereum mainnet and Rinkby testnet as well as the Polygon mainnet and Mumbai testnets.

Simulate external adapter calls

Use the simulator to make test calls to the external adapter and automatically generate code which can be pasted into a Solidity smart contract.
